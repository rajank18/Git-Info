How to use Git

By RAJAN

Step 1: Download Git

To start using Git, you need to download and install it on your computer. Choose the appropriate link below:

  • Download for Windows
  • Download for macOS
  • Download for Linux

Step 2: Configure Git

After installing Git, you need to configure your username and email. Run the following commands in your terminal:

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

Step 3: Initialize a Git Repository

Open your terminal or command prompt and navigate to your project directory. Run the following command to initialize a new Git repository:

git init

Step 4: Commit Your Changes

After making changes to your files, you need to commit them to your repository:

git add .
git commit -m "Your commit message"

Step 5: Push to GitHub

To push your changes to GitHub, you need to link your local repository to a remote repository. Run the following commands:

git remote add origin https://github.com/your-username/your-repo.git
git push -u origin master

Using Git GUI

If you don't want to write this comands then you can use the Git GUI interface to manage and create repository

  1. Open Git GUI from your application menu.
  2. Click on "Create New Repository" and choose your project directory.
  3. Make changes to your files and then click "Rescan" to see the changes.
  4. Stage the changes by selecting them and clicking "Stage Changed" button.
  5. Enter your commit message and click "Commit".
  6. To push your changes to GitHub, go to "Remote" menu, select "Add" and add your repository URL.
  7. Finally, click "Push" to upload your changes.

Update the repository

So, basically from this you can Update the repository by adding new files that you want to.

  1. Repeat Step 3(git init) in same folder for which you want to update repositroy.
  2. Repeat step 4.
  3. Repeat Step 5 but don't use "Origin" command.(As repository is already linked with folder.)
  4. And You are done. Your repositroy will be updated with new files.
